# FCFireCombat: Ultimate Survival

*Experience the next level of mobile combat!* Step into the battlefield of *FCFireCombat*, where only the fastest and smartest survive. This project is built using Unity and features high-fidelity graphics and intense strategic gameplay.

### 🎮 Game Features:
* *Massive Survival Map:* Vast environment with strategic loot locations.
* *Cinematic Graphics:* Ultra-realistic 4K textures and AAA-grade character designs.
* *Advanced Combat:* Realistic gun mechanics and 4v4 competitive modes.
* *Tactical Looting:* Features a 3-level backpack system and auto-loot logic.

### 🛠️ Technical Details:
* *Engine:* Unity 6 (6000.0.1f1)
* *Render Pipeline:* Universal Render Pipeline (URP)
* *Platform:* Android
